Estrutura básica do HTML

(Voltar)

A aula se resumiu em explicar a base de uma estrutura HTML contendo 3 elementos:

  1. Tag: HTML para descrição de comandos básicos do HTML.
  2. Head: Cabeçalho de informações que descreve itens que não irão aparecer ao Client como título da Aba por exemplo.
  3. Body: Todas as informações como títulos, textos e parágrafos que ao contrário do body, irão aparecer ao Client.

Atributos das Tags

(Voltar)

Na aula de atributos foi realizado a orientação de alguns comandos para identificação de estrutura, formatação de texto, entre outros. Abaixo alguns exemplos:

  • id: Identificador de módulos
  • style: Formatação de texto conforme orientação como cor da fonte por exemplo.
  • class: Categoriza as classes dentro da linha de código.
  • img: Coloca fotos na página HTML amarradas ao src e sinal de igual (=) possibilitando colocar links para buscar essas imagens.
    • width: Comando para acertar o dimensional da imagem colocada de acordo com a dimensão solicitada.

    Textos

    (Voltar)

    Textos são colocados através de parágrafos através do comando "P" e sendo entre os sinais, os dados que serão apresentados ao Client.

    Podem se classificar também como títulos e subtítulos sendo classificados de H1 até H6 tendo informações padrões de margem e texto

    Texto com sublinhado também com comando U que significa Underline.

    Texto como subtítulo também idetificado pelo comando sub.
